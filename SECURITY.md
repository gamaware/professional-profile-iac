# Security Policy

## Scope

This repository manages AWS infrastructure for a static website including S3,
CloudFront, Route 53, and ACM resources.

## Reporting a Vulnerability

If you find a security vulnerability in any Terraform configuration or CI/CD
workflow, please **do not open a public GitHub Issue**.

Instead, report it privately via email: <gamaware@gmail.com>

Include:

- The affected file or resource
- A description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact

## Response

Reports will be reviewed and addressed promptly.
